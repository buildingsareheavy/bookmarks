# bookmarks

A self-hosted personal bookmarking app built with vanilla JS and Node.js.

## Running Locally

`npm run dev`

### Note

`bruno-bookmarks/` contains a [Bruno](https://usebruno.com/) API collection for testing the server endpoints locally. Open Bruno, point it at this folder, and all the requests will be ready to use.


## Steps

- create class that handles the bookmark objects
- build components and putting them into their own files
- when building out the API always do it in this order: read, create, delete, update
- use switch case for the server, with everything as it's own files, instead of if else
- after this feels complete, add one small technology, like make a new branch and add express, or add typescript. But do those one at a time
